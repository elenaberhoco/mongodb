Un peu de maths¶

Vous pouvez utiliser des formules mathématiques \(y = f(x)\), voire même :

\[ y = f(x) \]

Un peu de code¶

De la même manière, vous pouvez avoir des morceaux de code y = f(x) au milieu de votre texte, ou bien :

def f(x):
    return 2 * x

print(f(2))

Lorsque vous voudrez présenter du code MongoDB, vous utiliserez la coloration syntaxique correspondant au langage javascript :

db.NYfood.find(
    {"cuisine": "Chinese", "borough": "Bronx"}
)

Vous remarquerez que ce code est mis en forme, mais pas exécuté.

On peut même imaginer mettre, en parallèle, la syntaxe MongoDB et son équivalent SQL :

db.t.find(
    {"a": 1},
    {"b": true}
)

SELECT b
FROM t
WHERE a = 1

ou encore :

MongoDB

db.t.find(
    {"a": 1},
    {"b": true}
)

Équivalent SQL

SELECT b
FROM t
WHERE a = 1

Malheureusement, dans ce cas de contenus disposés en onglets ou en “cartes”, on ne peut pas insérer de code exécutable dans ces contenus (voir plus bas).
